# 🛡️ lifecycle_guard_platform_interface

**The common interface for mission-critical background execution.**

This package defines the contract for `lifecycle_guard` across all platforms. It ensures a consistent API whether you're running on Android, iOS, or future platforms.

## 🚀 Purpose

This is a **Platform Interface** package. It is not intended to be used directly by app developers. App developers should use the main [lifecycle_guard](https://pub.dev/packages/lifecycle_guard) package.
